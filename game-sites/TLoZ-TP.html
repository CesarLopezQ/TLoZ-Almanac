The Legend of Zelda: Twilight Princess

The thirteenth game in the franchise, originally released for the Game Cube, later released to the Wii and remastered for the Wii U.

The game was the first in the series to be rated as T by ESBR due to its darker and more mature themes.

The gameplay pioneered the use of motion controls in Zelda games, allowing for basic point and click controlls, as well as some responsive movements to use the sword an shield (a system that we see fully developed in Skyward Sword).
